# CDL-03-demo-markdown
Demo for sesion 03 of CDL summer 2026# Hello,World! Assignement

Create a program that prints "Hello,Wordl!"

Also create a `Makefile` to build the program.

## Starting Point

Start from the following skeleton:

```c=
#include <stdio.h>

int main(void)
{
    ...
}
```

## Build
Build the program using:

```
make
```
## Test

To test, run:

```
console
./helloworld | grep 'Hello,World!'
```
